## About
This project displays a restaurant menu using a reusable `MenuItem` component and the `map()` method to render lists. The app includes sections for Main Courses and Desserts, restaurant info, and a footer.

## Components
- **MenuItem.jsx**: Reusable component that accepts props:
  - `name` - Dish name
  - `category` - Category (e.g., Main Course, Dessert)
  - `price` - Price of the item (number)
  - `description` - Brief description
  - `isVegetarian` - Boolean (if true shows "Vegetarian" badge)
